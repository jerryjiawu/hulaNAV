üîß System Configuration & Connection
connect(server_ip=None)
Establishes connection between host computer and drone. The host computer acts as server.

Parameters:

server_ip (Optional) Drone IPv4 address, auto-detected if not provided

Returns:

True: Success, False: Failure
# Connect to drone (auto-detect IP) drone = UserApi() if drone.connect(): print("‚úÖ Connected successfully!") else: print("‚ùå Connection failed")
‚úàÔ∏è Flight Control Methods
single_fly_takeoff(led=0) Essential
Real-time control for drone takeoff. Initiates flight mode and lifts drone from ground.

Parameters:

led LED configuration: {'r':0,'g':0,'b':0,'mode':1} - r,g,b: colors, mode: 1/steady, 2/off, 4/RGB cycle, 16/rainbow, 32/blink, 64/breathing
# Take off with blue LED drone.single_fly_takeoff({'r':0,'g':0,'b':255,'mode':1})
single_fly_touchdown(led=0) Essential
Real-time control for drone landing. Safely lands drone and stops propellers.

Parameters:

led LED configuration (same format as takeoff)
# Land with red LED drone.single_fly_touchdown({'r':255,'g':0,'b':0,'mode':32})
single_fly_curvilinearFlight(x, y, z, direction=True, led=0) Advanced
Curvilinear flight allows the drone to fly in smooth curved paths to specified coordinates. This enables predictable 3D curved trajectories perfect for visualization and path planning.

Parameters:

x x-axis coordinate (cm) - aircraft left/right, right is positive
y y-axis coordinate (cm) - aircraft forward/backward, forward is positive
z z-axis coordinate (cm) - aircraft up/down, up is positive
direction True: counterclockwise, False: clockwise (default True)
# Curved flight: 100cm forward, 50cm right, 20cm up, counterclockwise drone.single_fly_curvilinearFlight(50, 100, 20, True)
single_fly_forward(distance, led=0)
Real-time control for drone flying forward.

Parameters:

distance flight distance (cm)
drone.single_fly_forward(100) # Move 100cm forward
single_fly_back(distance, led=0)
Real-time control for drone flying backward.

Parameters:

distance flight distance (cm)

single_fly_straight_flight(x, y, z, led=0)
Direct straight flight to coordinates.

Parameters:

x, y, z coordinate positions (cm)
single_fly_radius_around(radius, led=0)
Radius orbit flight around current position.

radius orbit radius (cm, positive: counterclockwise, negative: clockwise)
single_fly_autogyration360(num, led=0)
Clockwise/counterclockwise rotation for specified number of turns.

Parameters:

num number of turns (positive: counterclockwise, negative: clockwise)
single_fly_somersault(direction, led=0)
Drone performs forward/backward/left/right roll in place.

Parameters:

direction 0=forward, 1=back, 2=left, 3=right
single_fly_bounce(frequency, height, led=0)
Real-time control for drone bouncing motion.

Parameters:

frequency number of bounces

single_fly_hover_flight(time, led=0)
Drone hover in place for specified duration.

time hover time (seconds)
single_fly_barrier_aircraft(mode)
Enable or disable obstacle avoidance system.

Parameters:

mode True: enable, False: disable
single_fly_Line_walking(fun_id, dist, way_color)
Line following detection and navigation.

Parameters:

fun_id 0: forward line following, ignore intersections
dist distance, unit cm
way_color line following color range, 0-black 255-white

Returns:

üëÅÔ∏è Computer Vision & AI Recognition
single_fly_AiIdentifies(mode) AI
Advanced AI-powered tag and marker identification system. Recognizes numbers, arrows, and letters.

Parameters:

mode 0-9: identify number tags, 10-13: arrows (left/right/up/down), 20: end task, 65-90: uppercase letters A-Z

Returns:

x, y, z: tag coordinates relative to drone
angle: angle between tag and drone
result: True/False for success/failure
# Identify number tag 5 result = drone.single_fly_AiIdentifies(5) if result['result']: print(f"Found tag at: {result['x']}, {result['y']}, {result['z']}")
single_fly_Optical_flow_alignment(qr_id, qr_size, angle=0)
Optical flow alignment with QR code for precise positioning.

Parameters:

qr_id target QR code id [0-9]
qr_size physical size of QR code, range [6, 30], unit: cm
angle align with QR code and rotate n degrees

Returns:

result: True/False for alignment success/failure
single_fly_Optical_flow_recognition(qr_id, qr_size)
Optical flow recognition of QR code with distance measurements.

Dictionary: {result, x, y, z, yaw, qr_id} - distances and angles to QR code

üìä Sensors & Status Monitoring
get_coordinate() Essential
Get current drone 3D coordinates for position tracking.

Returns:

[x, y, z] coordinate array
pos = drone.get_coordinate() print(f"Drone position: X={pos[0]}, Y={pos[1]}, Z={pos[2]}")
get_yaw()
Get drone orientation angles in degrees.

Returns:

[yaw angle, pitch angle, roll angle]
get_plane_distance() ToF
Get drone ToF (Time of Flight) height - precise ground-relative altitude measurement using laser/infrared sensors.

Returns:

integer: drone ToF height (cm) - accurate to within 1-3cm
height = drone.get_plane_distance() print(f"Height above ground: {height}cm")
get_battery()
Get drone battery percentage for flight safety monitoring.

Returns:

integer: battery percentage (0-100)
get_accelerated_speed()
Get drone acceleration measurements for motion analysis.

Returns:

[X, Y, Z] acceleration values
get_plane_speed()
Get drone body velocity in all three axes.

Returns:

[X, Y, Z] velocity values
get_plane_id()
Get unique drone identifier.

Returns:

integer: drone ID
Plane_getBarrier()
Get comprehensive obstacle avoidance sensor data.

Returns:

Dictionary: {'forward': True/False, 'back': True/False, 'left': True/False, 'right': True/False}
üîß Hardware Control & Accessories
single_fly_lamplight(r, g, b, time, mode)
Control drone LED lighting system with various effects.

Parameters:

r, g, b RGB color values (0-255)
time light duration (seconds)
mode 1=steady, 2=off, 4=RGB cycle, 16=rainbow, 32=blink, 64=breathing
# Rainbow effect for 5 seconds drone.single_fly_lamplight(255, 255, 255, 5, 16)
plane_fly_generating(type, data, reserve)
Control laser firing system for interactive applications.

Parameters:

type 0=single shot, 1=continuous fire, 2=enable receiver, 3=disable receiver, 4=unlimited fire, 5=disable firing
data continuous fire frequency (shots/second, range 1-14)
reserve ammunition count (range 1-255)
plane_fly_laser_receiving()
Detect laser hits on drone's receiver.

Returns:

True: hit detected, False: no hit
Plane_cmd_electromagnet(type)
Control external electromagnet accessory.

Parameters:

type 2=electromagnet attract, 3=electromagnet release
Plane_cmd_clamp(type, angle=0)
Control external clamp and electromagnet system.

Parameters:

type 0=clamp close, 1=clamp open, 2=clamp angle, 3=electromagnet attract, 4=electromagnet release
angle clamp rotation angle (0-180 degrees)
plane_fly_arm()
Start low speed rotation of propellers for system preparation.
plane_fly_disarm()
Stop low speed rotation of propellers.
üì∑ Camera & Media Operations
Plane_cmd_swith_rtp(type) Essential
Enable or disable video streaming from drone camera.

Parameters:

type 0=enable video stream, 1=disable video stream
# Enable video streaming drone.Plane_cmd_swith_rtp(0)
get_image_array()
Capture current frame from video stream as numpy array for processing.

Returns:

numpy.ndarray or None - current video frame
import cv2 frame = drone.get_image_array() if frame is not None: cv2.imshow('Drone View', frame)
Plane_fly_take_photo()
Capture still photo (requires active video stream).
Plane_cmd_switch_video(type=0)
Start or stop video recording.

Parameters:

type 0=start recording, 1=stop recording
single_fly_flip_rtp()
Open video stream display (requires enabled video stream).
Plane_cmd_camera_angle(type, data)
Adjust main camera pitch angle for different viewing perspectives.

Parameters:

type 0=up, 1=down (absolute), 2-3=algorithm control, 4=calibration, 5-6=block relative
data rotation angle (0-90 degrees)
Plane_cmd_switch_QR(type)
Enable or disable QR code positioning system.

Parameters:

type 0=enable positioning QR code, 1=disable positioning QR code
